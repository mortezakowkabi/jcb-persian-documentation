# بررسی عمیق قابلیت‌های JCB

پوشه **Features** مجموعه‌ای از راهنمایی‌های متمرکز را گردآوری می‌کند که قابلیت‌های کلیدی Joomla Component Builder را بسط می‌دهد. هر مقاله توضیح می‌دهد که چه زمانی از هر قابلیت استفاده کنید، چرا اهمیت دارد و چگونه مرحله‌به‌مرحله آن را پیکربندی کنید تا اعضای جدید تیم بتوانند با اطمینان مشارکت کنند. اگر نیاز به یک مسیر گام‌به‌گام و قابل دنبال کردن و مرور در بقیه مستندات دارید، از این نقطه با [راهنمای شروع کار](../README.md) همراه شوید.

## قابلیت‌های موجود

* [ویوی مدیریتی JCB](./JCB-Admin-Views.md) — داده‌های خود را یک بار مدل‌سازی کنید و اجازه دهید JCB تمام تجربه بخش ادمین Joomla را کامپایل کند.

برای دسترسی به فهرست و سایر آموزشها، به [فهرست اصلی مستندات](../index.md) بازگردید.
