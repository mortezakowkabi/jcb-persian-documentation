# بررسی عمیق قابلیت‌های JCB

پوشه **ویژگی‌ها** مجموعه‌ای از راهنمایی‌های متمرکز را گردآوری می‌کند که قابلیت‌های کلیدی Joomla Component Builder را بسط می‌دهد. هر مقاله توضیح می‌دهد که چه زمانی از هر قابلیت استفاده کنید، چرا اهمیت دارد و چگونه مرحله به مرحله آن را پیکربندی کنید تا اعضای جدید تیم بتوانند با اطمینان مشارکت کنند. اگر نیاز به یک مسیر گام به گام و قابل مرور در بقیه مستندات دارید، در این نقطه شروع با [راهنمای شروع کار](../README.md) همراه شوید.

## قابلیت‌های موجود

* [ویوهای مدیریتی JCB](./JCB-Admin-Views.md) — داده‌های خود را یک بار مدل‌سازی کنید و اجازه دهید JCB تمام تجربه بخش مدیریت Joomla را کامپایل کند.

برای دسترسی به لینکهای دیگر آموزشی، به [فهرست اصلی مستندات](../index.md) بازگردید.
